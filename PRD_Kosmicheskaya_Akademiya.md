## Визуальная система

### Персонажи Академии

5 персонажей закреплены за возрастными группами и направлениями. Используются как иллюстративные герои Hero-блоков, элементы навигации и маскоты внутри платформы.


> 📌 Распределение по рынкам: Для российского рынка в Hero и визуалах используются Лео, Аури, Арло. Для казахстанского — Лео, Лина (азиатская внешность соответствует казахстанской аудитории), Арло. Зара задействована на обоих рынках для демонстрации инклюзивности.

## Визуальная система

### Дизайн-система и стиль


**Цветовая палитра**

#1A1E35 — Космический фон (основной) #2DD4BF — Тил/бирюза (Кадеты, акцент) #FB923C — Оранжевый (Лео, CTA) #7C3AED — Фиолетовый (Стажёры, девочки) #1E3A5F — Тёмно-синий (Арло, Команда) #4ADE80 — Зелёный неон (детали костюмов)


**Типографика**

- H1: Nunito / Unbounded, 700, 38–52px, светло-белый
- H2: Nunito, 600, 24–32px
- Body: Onest / Nunito, 400, 15–16px
- Кнопки: 600, 15px, letter-spacing 0.02em
- Подписи: 12px, text3 color
- Минимальный размер текста для мобайла: 14px

**3D-стиль иллюстраций**

- Cartoon-3D рендер (стиль Pixar/Big Hero)
- Мягкое студийное освещение, без жёстких теней
- Персонажи на нейтральном или космическом фоне
- Частицы, звёзды, голограммы как декоративные элементы
- Анимации: float (левитация), glow, sparkle
- Все персонажи — один стиль и пропорции

**UI-компоненты**

- Rounded corners: 12–16px для карточек, 8px для элементов
- Border: 0.5px, rgba(255,255,255,0.1)
- Кнопка Primary: оранжевый/бирюзовый fill, белый текст
- Кнопка Secondary: transparent, border, белый текст
- Badges: полупрозрачные pill-кнопки
- Hover: scale(1.02) + glow-эффект
- Анимация появления: fade-up 0.4s ease-out

## Архитектура

### Воронка и структура сайта


> Схема воронки (3 уровня) Уровень 1 → Уровень 2 (×3 группы) → Уровень 3 (×6 курсов). Всего: 1 + 3 + 6 = 10 страниц на рынок. Плюс Thank You Page, страница «Все работы», About-страница = ~14 страниц на рынок, ~28 итого при раздельных доменах/поддоменах.


**Уровень 1 — Главная**

1 страница. Холодная аудитория. Цель: заинтересовать → направить к выбору группы. Минимальная цель: захват контакта через квиз.


**Уровень 2 — Группы**

3 страницы (Кадеты / Стажёры / Команда). Тёплая аудитория. Цель: убедить, что подходит → направить к выбору направления.


**Уровень 3 — Курсы**

6 страниц (3 группы × 2 направления). Горячая аудитория. Цель: закрыть все возражения → оплата / заявка.

### Блок 01 — Шапка (Header)

*Sticky навигация · Присутствует на всех страницах*

**Тип:** Ключевой


#### Описание


**Назначение**

Постоянная навигация, доступная с любой точки страницы. Обеспечивает мгновенный доступ к главному CTA и контактам независимо от позиции скролла.


**Маркетинговое обоснование**

Sticky-позиция удерживает кнопку «Бесплатный урок» в зоне видимости 100% времени. Для холодного трафика из соцсетей наличие телефона/Telegram в шапке создаёт ощущение реальной компании. Разделение на роли через меню снижает когнитивную нагрузку.


**Состав элементов**

- Логотип (текст + иконка ракеты/звезды) — ссылка на главную
- Навигация: «Академия» / «Группы ▾» / «Курсы» / «О нас» / «Цены»
- Выпадающее меню «Группы»: Кадеты 7–9 / Стажёры 10–13 / Команда 14+
- Телефон 8-800-XXX-XX-XX (бесплатный)
- Иконки: Telegram + WhatsApp (ссылки на чаты)
- Кнопка Primary: «Бесплатный урок»
- Кнопка Secondary: «Войти» (для действующих учеников)

#### Тексты RU


**Пункты навигации**


**Навигация**

Академия · Группы · Курсы · О нас · Цены

Dropdown «Группы»: Кадеты (7–9 лет) · Стажёры (10–13 лет) · Команда (14+)


**CTA в шапке**

→ `[КНОПКА: Бесплатный урок]`
→ `[КНОПКА: Войти]`

**Слоган под логотипом (опционально)**

Обучай ребёнка создавать с ИИ, а не зависеть от него


#### Визуал


**Описание визуала**

Высота 64–68px на десктопе, 56px на мобайле. Фон: полупрозрачный тёмно-космический (#0e1118, opacity 0.92) + backdrop-blur(12px). Логотип слева: иконка ракеты (24px, цвет var(--teal)) + текст «Космическая Академия» (Unbounded, 500, 15px). Навигация по центру или слева от CTA. Телефон серый (#8b96b0). CTA-кнопка: border-radius 8px, background gradient teal→orange, белый текст. Мобайл: только логотип + кнопка + бургер.


#### Технические требования

- position: sticky; top: 0; z-index: 1000
- Анимация появления: slideDown 0.3s при загрузке страницы
- При скролле вниз >100px: добавляется border-bottom 0.5px
- Выпадающее меню: hover (десктоп) / tap (мобайл), анимация fadeDown 0.2s
- CTA «Бесплатный урок»: ведёт на якорь #quiz (квиз) на L1; на L3 — на #form
- Мобайл <768px: бургер-меню, full-screen overlay с теми же пунктами
- Telegram/WhatsApp: открываются в новой вкладке, deep link на чат поддержки

### Блок 02 — Hero-блок — первый экран

*Главный конверсионный блок · 100vh · Персонаж Лео + порталы групп*

**Тип:** Критический


#### Описание


**Назначение**

Первый экран, который видят 100% посетителей. Задача — за 4–6 секунд ответить на вопрос «что это и зачем мне?» и направить к выбору возрастной группы.


**Структура блока**

- Левая часть (60% ширины): H1 + подзаголовок + 2 CTA-кнопки + бейджи доверия
- Правая часть (40%): 3D-персонаж Лео, левитирующий с анимацией
- Нижняя часть: 3 карточки-портала по возрастным группам
- Фон: анимированное звёздное небо, туманности (CSS particles или WebGL-легкий)

**Бейджи доверия под CTA**

«150+ учеников» · «4 формата обучения» · «Первый урок бесплатно»


#### Тексты RU


**Вариант A — для «Тревожных» (основной)**


**H1 · основной заголовок**

*Ваш ребёнок научитсяуправлять ИИ —а не зависеть от него*

Космическая Академия — онлайн-школа, где дети от 7 до 17 лет создают настоящие продукты с помощью ИИ: книги, игры, сайты, мультики. Не просто учатся — делают.

→ `[КНОПКА: Выбрать курс для ребёнка]`
→ `[КНОПКА: Смотреть бесплатный урок]`

**Вариант B — для «Прагматиков»**


**H1**

*ИИ-навык, которыйостанется с ребёнкомнавсегда*

Мы не учим конкретным нейросетям — мы учим принципам. После курса ребёнок создаёт реальный продукт от идеи до результата. Для детей 7–17 лет.

→ `[КНОПКА: Узнать про курс]`

**3 портала — тексты карточек**


**Карточка 1 · Кадеты**

**Кадеты · 7–9 лет**

«Творцы» — учатся оживлять идеи: пишут сказки, создают иллюстрации, делают открытки с помощью ИИ


**Карточка 2 · Стажёры**

**Стажёры · 10–13 лет**

«Исследователи» — используют ИИ для учёбы и хобби: создают презентации, исследования, творческие проекты


**Карточка 3 · Команда**

**Команда · 14+ лет**

«Архитекторы» — разрабатывают реальные продукты: игры, сайты, приложения с нуля до запуска


#### Визуал


**Визуальное описание**

Тёмный космический фон с анимированными звёздами (CSS keyframes или Lottie). Персонаж Лео справа — 3D-рендер, PNG с прозрачным фоном, анимация float (translateY ±12px, 3s ease-in-out, infinite). Фиолетово-голубые туманности в фоне — декоративный элемент CSS radial-gradient. Три портала снизу — горизонтальный ряд карточек: каждая с уникальным цветом (teal / purple / orange-dark), иконкой персонажа и hover-эффектом подъёма.


**Лео — основной герой Hero L1**
Поза: указывает пальцем на заголовок или вверх. Файл: leo-hero-front.png, минимум 600×900px @2x. Анимация float на лендинге — CSS, не GIF (производительность).

#### Технические требования

- Высота: min-height: 100vh; padding-bottom: 80px
- Фон-анимация: CSS particles или tsParticles (лёгкая библиотека, <30KB gzip)
- Персонаж: lazyload, WebP + PNG fallback, srcset для @2x
- Порталы: display grid 3 columns; gap 16px; на мобайле — stack
- CTA «Выбрать курс»: scroll to #portals (якорь на тот же блок порталов)
- A/B тест: заголовок варианта A vs B; победитель через 2 недели по CTR на порталы
- LCP оптимизация: Hero-персонаж — priority load, preconnect шрифты
- Анимация появления: stagger — H1 0ms, подзаголовок 150ms, кнопки 300ms, персонаж 100ms
